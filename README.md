WorkSphere - Gestion Interactive du Personnel


Application web moderne pour la gestion visuelle et interactive du personnel dans les espaces de travail.


Description du Projet

WorkSphere est une solution innovante permettant aux entreprises de gérer efficacement la répartition de leurs employés sur un plan d'étage interactif. 
L'application offre une interface intuitive pour visualiser, organiser et gérer les affectations du personnel tout en respectant les contraintes de sécurité et d'accès.


Fonctionnalités Principales

Gestion des Employés

Ajout d'employés avec formulaire complet (nom, rôle, photo, email, téléphone)
Gestion dynamique des expériences professionnelles
Validation des données avec expressions régulières
Prévisualisation de la photo de profil
Affichage détaillé du profil employé

Gestion Spatiale

Plan d'étage interactif avec 6 zones distinctes
Salle de conférence
Réception
Salle des serveurs
Salle de sécurité
Salle du personnel
Salle d'archives

Système de restrictions par rôle
Indicateurs visuels pour les zones vides obligatoires
Limitation du nombre d'employés par zone

Règles d'Accès

Réceptionnistes : Accès uniquement à la Réception
Techniciens IT : Accès uniquement à la Salle des serveurs
Agents de sécurité : Accès uniquement à la Salle de sécurité
Manager : Accès à toutes les zones
Nettoyage : Accès partout sauf Salle d'archives
Autres rôles : Accès libre sauf zones restreintes

Technologies Utilisées

HTML5 : Structure sémantique
CSS3 : Styles personnalisés avec CSS Grid et Flexbox
Tailwind CSS : Framework CSS utilitaire
JavaScript: Logique applicative
Git & GitHub : Gestion de versions

Fonctionnalités Détaillées

Validation des Données

Nom : 3 à 10 caractères alphabétiques
Téléphone : Format marocain (+212 ou 0) suivi de 6/7 et 8 chiffres
Email : Format standard (exemple@domaine.com)
Dates : Validation que la date de début précède la date de fin


Interface Utilisateur

Design moderne avec palette de couleurs cohérente
Animations CSS fluides
Notifications toast pour les actions utilisateur
Interface responsive (mobile, tablette, desktop)

Responsive Design

Tablette Portrait : 768px - 1023px
Tablette Paysage : 1024px - 1279px
Desktop : 1280px et plus
